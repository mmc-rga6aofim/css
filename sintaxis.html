Sintaxis

El objetivo fundamental del lenguaje CSS (Cascading Style Sheets) es permitir que el navegador pinte elementos de una página con características específicas, como colores, posición o decoración. La sintaxis de CSS refleja estos objetivos y consta de los siguientes bloques básicos de construcción:

See the Pen Sintaxis CSS by Alan Rodríguez Gallegos (@Alan-Rodr-guez-Gallegos) on CodePen.





Selector

Es una expresión que selecciona uno o varios elementos HTML a los que se les aplicarán los estilos. Puede ser el nombre de una etiqueta HTML (como "h1" o "p"), una clase (precedida por un punto, como ".mi-clase"), un ID (precedido por un símbolo de numeral o almohadilla, como "#mi-id"), un atributo, una pseudo-clase o incluso una combinación de estos elementos.

Propiedad

Es el atributo específico que se desea estilizar en los elementos seleccionados. Las propiedades pueden ser diversas, como "color" (color del texto), "font-size" (tamaño de la fuente), "background-color" (color de fondo), "margin" (margen), "padding" (relleno) y muchas más. Cada propiedad tiene un propósito y efecto específico sobre los elementos.

Valor

Es el valor asignado a la propiedad seleccionada. Los valores pueden ser colores (como "red" o "#00ff00"), medidas (como "10px" o "2em"), números, cadenas de texto, imágenes, entre otros. El valor determina cómo se verá y se comportará la propiedad en los elementos seleccionados.
