Conceptos básicos

Selectores

Los selectores en CSS permiten identificar los elementos HTML a los que se les aplicarán estilos. Algunos ejemplos de selectores son: selectores de tipo, selectores de clase, selectores de ID y selectores de atributo.

Selectores básicos

  • Selectores de tipo (etiquetas):


    Este selector se utiliza para seleccionar elementos basados en su etiqueta HTML. Puedes utilizar el nombre de la etiqueta como selector para aplicar estilos a todos los elementos de ese tipo. Por ejemplo, si deseas aplicar estilos a todos los párrafos (p) de tu página, puedes usar el selector p. Esto afectará a todos los elementos de párrafo en tu documento.

  • Selectores de clase:


    El selector de clase te permite seleccionar elementos que tienen asignada una clase específica en su atributo class. Puedes utilizar el prefijo . seguido del nombre de la clase para seleccionar los elementos deseados. Por ejemplo, si tienes un elemento de div con la clase "salte-del-salón", puedes seleccionarlo utilizando el selector .salte-del-salón. Esto aplicará estilos solo a los elementos que tengan la clase "salte-del-salón".

  • Selector de ID:


    El selector de ID se utiliza para seleccionar un elemento específico basado en su atributo id. Puedes utilizar el prefijo # seguido del nombre del ID para seleccionar el elemento deseado. Por ejemplo, si tienes un elemento de h1 con el ID "principal", puedes seleccionarlo utilizando el selector #principal. Esto aplicará estilos solo a ese elemento en particular.

  • Selector de atributo (corchetes):


    El selector de atributo te permite seleccionar elementos basados en un atributo específico. Puedes seleccionar elementos que tengan ese atributo en general ([atributo]) o elementos que tengan un atributo con un valor específico ([atributo="valor"]). Por ejemplo, puedes seleccionar todos los elementos input que tengan el atributo type establecido en "text" utilizando el selector [type="text"].

  • Selector universal:


    El selector universal CSS (*) coincide con elementos de cualquier tipo.

Combinadores

  • Combinador de descendencia (solo espacio):


    Este combinador selecciona elementos secundarios que están anidados dentro de un elemento padre. El selector consiste en especificar el selector del elemento padre seguido de un espacio y luego el selector del elemento secundario. Por ejemplo, si deseas seleccionar todos los elementos p que son descendientes directos de un div, puedes utilizar el selector div p. Esto aplicará estilos a todos los párrafos que están anidados dentro de un elemento div.

  • Combinador de hijo directo (>):


    Este combinador selecciona elementos que son hijos (se le llama hijo a un elemento cuando está dentro de otro elemento, el cual se menciona como padre) directos de un elemento padre. Utilizas el selector del elemento padre seguido del símbolo > y luego el selector del hijo directo. Por ejemplo, si deseas seleccionar todos los elementos li que son hijos directos de un ul, puedes utilizar el selector ul > li. Esto aplicará estilos solo a los elementos li que son hijos directos de un elemento ul.

  • Combinador de adyacencia (+):


    El combinador de adyacencia en CSS se representa con el símbolo "+" y se utiliza para seleccionar el siguiente elemento adyacente que sigue inmediatamente al primer elemento especificado. En otras palabras, selecciona un elemento hermano que está justo después del primer elemento en el árbol de elementos HTML.

  • Combinador de hermano general (~):


    Este combinador selecciona todos los elementos hermanos que siguen al primer elemento, sin importar su posición. Se utiliza el símbolo ~ entre los selectores.

Propiedades

Las propiedades de CSS definen cómo se verán los elementos seleccionados. Hay una amplia gama de propiedades, como color de fondo, tamaño de fuente, margen, relleno, etc. Algunos ejemplos de propiedades son: color, font-size, margin, padding, etc. A continuación se muestran algunas de las propiedades más comunes:

Color

Esta propiedad establece el color del texto de un elemento. Puedes especificar el color utilizando nombres predefinidos como "red" o "blue", valores hexadecimales como "#ff0000" para rojo, o valores RGB como "rgb(255, 0, 0)".

Font Size

Esta propiedad define el tamaño de la fuente de un elemento. Puedes especificar el tamaño utilizando unidades como píxeles (px), porcentajes (%), em (em), o puntos (pt).

Margin

Esta propiedad define los márgenes externos de un elemento, es decir, el espacio alrededor del elemento. Puedes especificar los márgenes utilizando valores individuales para cada lado (arriba, derecha, abajo, izquierda), o un solo valor para aplicarlo a todos los lados.

Padding

Esta propiedad define los márgenes internos de un elemento, es decir, el espacio dentro del borde del elemento. Al igual que con margin, puedes especificar los rellenos utilizando valores individuales o un solo valor para aplicarlo a todos los lados.
