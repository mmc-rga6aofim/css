Animaciones y transiciones

Animaciones

Las animaciones en CSS son una forma de agregar movimiento y cambios visuales a los elementos de una página web. Puedes hacer que un elemento se mueva, cambie de tamaño, cambie de color, entre otras cosas.

@keyframes

La regla @keyframes en CSS se utiliza para definir los diferentes momentos clave de una animación. Es una parte fundamental para crear animaciones en CSS.

Cuando usas @keyframes, especificas los estados de la animación en porcentajes o palabras clave como from y to. Cada estado representa cómo quieres que se vea el elemento en ese momento específico de la animación.

Puedes definir múltiples estados clave dentro de @keyframes, lo que te permite controlar con precisión cómo se animarán las propiedades del elemento a lo largo del tiempo.

See the Pen keyframes by Alan Rodríguez Gallegos (@Alan-Rodr-guez-Gallegos) on CodePen.


En este ejemplo, @keyframes se llama mover y tiene dos estados clave. En el estado inicial (0%), se establece la propiedad left en 0, lo que significa que el cuadro se encuentra en la posición más a la izquierda. En el estado final (100%), se establece la propiedad left en 200px, lo que hace que el cuadro se mueva 200 píxeles hacia la derecha.

Después de definir @keyframes, puedes utilizar el nombre de la animación (mover en este caso) en la propiedad animation-name para aplicarla a un elemento y lograr el efecto de animación deseado.

@keyframes te brinda flexibilidad y control sobre cómo se desarrolla una animación en CSS, ya que puedes especificar los momentos clave y los cambios en las propiedades de manera precisa a lo largo del tiempo.

Propiedades animadas

Las propiedades animadas en CSS son las propiedades de estilo que puedes modificar en los diferentes estados clave de una animación para crear efectos visuales dinámicos y de movimiento. Estas propiedades pueden incluir:



  • Posición y dimensiones:


    "top", "right", "bottom", "left": controlan la posición del elemento.

    "width" y "height": ajustan el ancho y alto del elemento.

  • Estilos visuales:


    "color", "background-color": cambian el color del texto o del fondo.

    "opacity": controla la opacidad del elemento.

    "border-color", "border-width", "border-radius": modifican el color, ancho y borde redondeado del elemento.

    "box-shadow": crea una sombra alrededor del elemento.

  • Transformaciones:


    "transform": permite aplicar transformaciones 2D y 3D al elemento, como rotación, escala y traslación.

  • Interacciones del usuario:


    "cursor": define el estilo del cursor cuando se coloca sobre el elemento.

    "pointer-events": controla la interacción con el elemento mediante el ratón.





animation-name

La propiedad animation-name en CSS se utiliza para darle un nombre a una animación que has definido previamente utilizando @keyframes. Es como ponerle un título a la animación.

Cuando defines una animación con @keyframes, especificas cómo quieres que se vea un elemento en diferentes momentos clave. Luego, con animation-name, le dices al elemento que aplique esa animación en particular.

Recuerda que es importante asegurarte de que el nombre de la animación en animation-name coincida exactamente con el nombre que has definido en @keyframes para que la animación funcione correctamente.





Otras propiedades de animación:

Hay varias propiedades adicionales que se pueden utilizar para controlar y ajustar la animación en CSS. Estas propiedades incluyen:

  • animation-duration:


    Esta propiedad establece la duración de la animación, es decir, cuánto tiempo tarda en completarse una repetición de la animación. Se especifica en segundos (s) o milisegundos (ms).

  • animation-timing-function:


    Esta propiedad define la función de temporización que determina cómo se distribuye la animación a lo largo de su duración. Puedes utilizar diferentes funciones como ease (predeterminado), linear, ease-in, ease-out, ease-in-out, entre otras.

  • animation-delay:


    Esta propiedad establece un retraso antes de que comience la animación. Puedes especificar un valor en segundos (s) o milisegundos (ms). Es útil para controlar el momento en que una animación se activa.

  • animation-iteration-count:


    Esta propiedad determina cuántas veces se repetirá la animación. Puedes establecer un número finito (por ejemplo, 3 para repetir la animación tres veces) o utilizar el valor infinite para repetir la animación indefinidamente.

  • animation-direction:


    Esta propiedad define la dirección de la animación. Puedes establecerla como normal (de inicio a fin), reverse (de fin a inicio), alternate (se invierte en cada repetición) o alternate-reverse (inicia en la dirección inversa y se invierte en cada repetición).

Estas propiedades adicionales te brindan mayor control sobre la apariencia y el comportamiento de la animación en CSS. Puedes ajustar la duración, el retraso, la función de temporización, la repetición y la dirección para crear efectos animados más personalizados y sofisticados.









Transiciones

Las transiciones en CSS permiten crear cambios suaves y fluidos entre los diferentes estados de un elemento, como al cambiar propiedades como el color, tamaño, posición, entre otros. Proporcionan una forma elegante de animar y mejorar la experiencia visual de una página web.

Propiedades de transición

  • transition-property:


    Especifica las propiedades que se animarán durante la transición.

  • transition-duration:


    Establece la duración de la transición, es decir, cuánto tiempo tarda en completarse.

  • transition-timing-function:


    Controla la forma en que se acelera o desacelera la transición.

  • transition-delay:


    Define un retraso antes de que comience la transición.





Transiciones en pseudoclases:

Las transiciones en pseudoclases son una forma de aplicar efectos de transición a un elemento cuando ocurre una interacción específica con él. Estas pseudoclases se utilizan para seleccionar y estilizar elementos en ciertos estados o situaciones.

  • :hover


    Esta pseudoclase se activa cuando el cursor se coloca sobre un elemento. Puedes utilizarla para animar propiedades como el color de fondo, el tamaño o la opacidad al pasar el cursor sobre el elemento.

  • :focus


    Esta pseudoclase se activa cuando un elemento obtiene el enfoque, generalmente mediante un clic o cuando se selecciona mediante la navegación por teclado. Puedes utilizarla para aplicar transiciones a elementos interactivos como formularios.

  • :active


    Esta pseudoclase se activa cuando se presiona un elemento, como un botón. Puedes utilizarla para aplicar transiciones a elementos mientras se mantenga presionado.
